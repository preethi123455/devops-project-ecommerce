## Product-Image Mapping
The products.ts file contains a precise mapping where:
- Each product ID is paired with its specific image file
- Images are semantically relevant to the product
- Multiple products can share related images (e.g., all t-shirts might use tshirt.jpg)
- Variety ensures visual interest when browsing categories
